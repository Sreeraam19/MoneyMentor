# MoneyMentor 💰

MoneyMentor is a personal finance management web application built using Django.

## Features

* User Authentication
* Income Tracking
* Expense Tracking
* Financial Analytics
* Dashboard Overview
* Responsive Interface

## Tech Stack

* Python
* Django
* PostgreSQL
* HTML
* CSS
* JavaScript

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/MoneyMentor.git
```

### Move into Project

```bash
cd MoneyMentor
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Server

```bash
python manage.py runserver
```

Visit:

```text
http://127.0.0.1:8000/
```
